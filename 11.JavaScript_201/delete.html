Deletion

The delete keyword can be used to delete properties but not variables themselves. 

"use strict";

const obj = { foo: 10, bar: 20 };

delete obj;
// The above command should throw an error. Without strict mode this would
// silently fail.

delete obj['foo'];

console.log(obj);
// Expected output: { bar: 20 }

Note: "use strict" does not work properly in the browser console.
